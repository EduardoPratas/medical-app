# Medical Appointment System

A Spring Boot demonstration project for managing patients, built with Java 17, Spring Boot, JPA/Hibernate, and Thymeleaf.

## 🚀 Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Data JPA / Hibernate
- Thymeleaf
- H2 Database (in-memory)
- Maven

📊 H2 Console
URL: http://localhost:8080/h2-console
JDBC URL: jdbc:h2:mem:medicaldb
